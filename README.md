# SpaceX Rockets App

SpaceX Rockets App is an iOS application built using Swift and UIKit.  
The app displays a list of SpaceX rockets and shows detailed information about each rocket.

## Features

- Displays SpaceX rockets in a UITableView
- Shows rocket name, first flight, cost, country, success rate, and description
- Supports navigation from rocket list screen to rocket detail screen
- Displays rocket detail images using UIScrollView
- Uses UIPageControl for image paging
- Built using MVC architecture
- Storyboard-based UI implementation

## Screens

### Rocket List Screen

The list screen displays rockets using a custom UITableViewCell.

Each cell contains:

- Rocket image
- Rocket name
- First flight date
- Cost
- Country
- Success rate
- Description

### Rocket Detail Screen

The detail screen shows selected rocket information.

It includes:

- Multiple rocket images
- Page control
- Rocket name
- First flight
- Country
- Success rate
- Mass
- Description

## Technologies Used

- Swift
- UIKit
- Storyboard
- UITableView
- Custom UITableViewCell
- UIScrollView
- UIPageControl
- MVC Architecture
